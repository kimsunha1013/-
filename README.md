# 서나의 먹부림 룰렛 🎲

## 배포 방법 (Vercel — 무료)

### 방법 A: GitHub 경유 (추천)
1. [github.com](https://github.com) 에서 새 repository 만들기
2. 이 폴더 전체를 업로드 (or `git push`)
3. [vercel.com](https://vercel.com) 접속 → GitHub 로그인
4. "Add New Project" → 방금 만든 repo 선택
5. 설정 건드릴 거 없음 → **Deploy** 클릭
6. 완료! 링크 공유하면 끝

### 방법 B: Vercel CLI (터미널)
```bash
npm install -g vercel
cd 이폴더
vercel
```

## 로컬 실행
```bash
npm install
npm start
```
